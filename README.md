# Chirality-Framework
This is a framework for meta-learning and knowledge generation. When expressed through its modalities it says: {problem statement} → Systematic → Process → Epistemic → Process → Epistemic → Alethic → Epistemic → Alethic → {consistency check}
This is the same general learning framework used by ReasonFlux to use reasoning traces grounded in a problem statement to enhance accuracy in math problems.  That same reinforcement learning framework is compatible with the Chirality Framework.

https://github.com/Gen-Verse/ReasonFlux

Distinct to the Chirality Framework is that it generalizes the process, so it is domain agnostic, and systme agnostic, and self-referential (it can build itself).

This permits the procedural generation of task-specific templates at critical steps in the reasoning process.  
This guides the LLM to answer along the "semantic valley" of the structured reasoning process. 

The result is a highly automated system for test-time scaling of LLM answers and generation of reasoning traces for reinforcement learning.
